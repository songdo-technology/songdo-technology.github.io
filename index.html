Google Calendar

  1. Click "+" next to "Other calendars"
  2. Select "From URL"
  3. Paste the subscription URL
  4. Click "Add calendar"

iPhone/iPad

  1. Settings → Calendar → Accounts
  2. Add Account → Other
  3. Add Calendar Subscription
  4. Enter the subscription URL

Mac Calendar

  1. File → New Calendar Subscription
  2. Enter the subscription URL
  3. Click "Subscribe"
  4. Choose your preferences

Outlook

  1. Add calendar → Subscribe from web
  2. Paste the subscription URL
  3. Name your calendar
  4. Click "Import"
